Openbroadcaster - Remote Installation Instructions

1. See dependencies.txt for remote dependencies

2. Run bash "obremote" or bash "obremote_loop -f". "bash obremote-h" for help

3. Http admin panel http://<IP_of_ Remote_Device>:23233 default user = admin default password = admin

4. Configure tabbed menus

5. Restart the remote

More info at http://wiki.openbroadcaster.com/
